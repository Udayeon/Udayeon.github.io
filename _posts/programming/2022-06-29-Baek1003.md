# 3. 풀이
C++ 함수를 파이썬으로 바꿔봄.
```py
def fibonacci(n):
    if n == 0:
        print("0")
        return 0
    elif n==1:
        print("1")
        return 1
    else:
        return fibonacci(n-1) + fibonacci(n-2)
```
```py

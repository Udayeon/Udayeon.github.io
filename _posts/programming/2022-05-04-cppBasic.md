# 3. 제어문 
## 3.1. 선택문

### 3.1.1. if문
주어진 조건을 만족하는 경우에만 특정 문장을 수행한다.

```c++
문장 1;
if(조건문) {
  문장2;
}
문장4
```
문장1을 수행하고 if문의 조건을 검사해서 참이면 {}안의 문장2 수행, 거짓이면 문장3으로 건너띔   
이때, {}안의 문장이 2개 이상일 경우에만 {}으로 묶고 한 개인 경우는 {}없이 걍 쓴다.

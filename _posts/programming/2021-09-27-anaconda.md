# 과정
* * *
리눅스 버전의 [아나콘다를 다운로드](https://www.anaconda.com/products/individual#linux) 받고 다운로드 받은 폴더에서
터미널을 열어 bash쉘로 실행

```
$ bash Anaconda3-2021.05-Linux-x86_64.sh
```
그리고나서 enter를 계속 눌러주면 알아서 설치됨

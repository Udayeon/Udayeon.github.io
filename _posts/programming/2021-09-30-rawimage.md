# 문제상황
* * * 
이미지 처리를 위해 파이썬에서 raw image를 읽어야 함.

# 해결
* * *
```py
fd=open('lena_water_mark.raw', 'rb')

rows = 512
cols = 512

f = np.fromfile(fd, dtype=np.uint8,count=rows*cols)
img = f.reshape((rows, cols)) #notice row, column format
fd.close()
```
